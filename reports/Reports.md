Project Title

Linux User and Process Management Toolkit

Name: Sowmiya R
Course: Bachelor of Computer Applications (BCA)
Year: III Year

Project Objective

The objective of this project is to develop a menu-driven Linux toolkit using Bash shell scripting to perform common Linux administration tasks such
as file permission management, user management, process management, and background job management.

Tools and Technologies Used

- Ubuntu Linux
- Bash Shell
- VirtualBox
- Nano Text Editor
- Linux Terminal

Modules Implemented

- Permission Management
- Ownership Management
- Group Management
- Display Current Umask
- User Creation
- Group Creation
- Process Monitoring
- Process Termination
- Nice/Renice Operations
- Background Job Management

Features

- Interactive menu-driven interface
- File permission management
- User and group management
- Process monitoring and termination
- Nice/Renice operations
- Background job management
- Input validation and error handling

Testing

All modules were tested with valid and invalid inputs. The toolkit performed the required operations correctly and displayed appropriate success or
error messages.

Result

The Linux User and Process Management Toolkit was successfully developed and tested. The project meets the required objectives and provides an easy
way to perform common Linux administration tasks.

Conclusion

This project helped in understanding Bash shell scripting, Linux commands, user management, process management, and file administration. It also 
improved practical knowledge of Linux system administration
